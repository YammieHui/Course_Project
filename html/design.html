Header and Footer

Header tag is used to group the logo and the company name, while the logo is inserted by an image tag.

An unordered list is created for later styling of the navigation bar and links are inserted by anchor tags for navigation purpose.

Footer tag is used to include the copy right information and the links to Home and Design pages.

Home (Index) Page

Item (d): Single movie information and its trailer

The block is separated into two parts, first by a division element with movie information, and second by a video element provided with two video formats and controls attribute to allow control of video's play.

Item (e): List of all movies

Table elements and unordered list elements are used for better layouting while images are inserted by image tags.

Now Showing Page

Item (f): Select a cinema

A dropdown list is created by select and option tags for user to select a cinema for buying ticket, also will be used for directing to the specified form in later CW.

Item (g): List of all movies on show now and Submit button

Separated forms represented one cinema each which include thumbnail of the movie (by image tag), dropdown lists showing movie session detail (by select and option tags) and submit button that will direct user to the ticket page (by action attribute in form tag).

Item (h): Cinema location

Map images are inserted by image tags.

Ticket Page

Item (j): Movie information headings

Multiple h4 elements included in a form which are for pre-setting.

Item (k): Seat plan of house

Item (l): Confirm button

A sumbit button is created that will send out values and direct user to the print page by action attribute in form tag.

Print Page

Item (m): Ticket information

Sub-divisions and headings are included to separate inforamtion of different tickets.

CW2

Overall theme layout

Number of style sheets & groups of style

Total 2 style sheets and 4 groups of style are used.

Style sheet

  • Overall layout and theme (common.css):
  • This style sheet is for sharing of common layout and theme between html pages. For example, header, footer and navigation are the shared elements with a fixed style and layout in all of my web page designs. Other features like buttons and movie/session information blocks also exist a common design. For the purpose of convenient maintenance, it is created as a separated style sheet and linked to every html pages.
  • For printing purpose in Print Page (print.css):
  • To enable a different layout style during print preview and printed document, a style sheet with media attritube and print value is linked to the Print Page.

Groups of style (Embedded style)

All embedded style groups are for page specific designing purpose. For example respective unclickable link in navigation bar.

  • Home Page (index.html):
  • For page specific design such as special alignment of movie lists (requirement no. 9), mouse-over effect on thumbnail (requirement no. 10), bold text (requirement no. 11) and video size etc.
  • Now Showing Page (showing.html):
  • For page specific design such as special alignment of Buy Ticket information (requirement no. 12), resetting margin for Cinema Select block etc.
  • Ticket Page (ticket.html):
  • For page specific design such as alignment of form's fieldsets, seat plan styling (requirement no. 13) and special hover effect (requirement no. 14) for seat selection etc.
  • Print Page (print.html):
  • For page specific design of ticket information's alignment.

CSS3 features

    Used in style sheet common.css, line 17 and 51.

  2. Box shadow:

    Used in style sheet common.css, line 35, 122 and 173.

Types of event handlers used

Setup in HTML

  • Home Page (index.html):
  • line 149 in body tag - Function calls of movieList() and firstPlay() when the page is loaded.
  • line 162 in video tag - Function call of setTimeout(autoVideo, 2000) once the video has ended.
  • Now Showing Page (showing.html):
  • line 123 in input tags - Function calls of enableData() when the submit button is clicked.
  • line 147 in body tag - Function call of selectedList(-1) when the page is loaded.
  • line 162 in select tag - Function call of selectedList(this.value) when the value of the select tag is changed.
  • Ticket Page (ticket.html):
  • line 222 in body tag - Function call of fillVar() and init() when the page is loaded.
  • line 277 in input tag - Function call of enableData() when the submit button is clicked.
  • Print Page (print.html):
  • line 116 in body tag - Function call of countTicket() when the page is loaded.

Setup in Javascript

  • Home Page (index.html):
  • line 123 in autoVideo() function - For recalling setTimeout(autoVideo, 2000) once the last video has ended.
  • Ticket Page (ticket.html):
  • line 166 in init() function - For calling booked() function when the submit button is clicked.
  • line 178 in booked() function - For removing the event handler when the seat is clicked.
